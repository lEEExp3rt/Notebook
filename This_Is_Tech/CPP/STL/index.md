# Standard Template Library 

> [!abstract] 本节摘要
> C++标准模板库 STL 实用内容收录

C++ 的标准模板库(*Standard Template Library*)是 C++ 标准库的核心组成部分，提供了一套通用的模板类和函数，用于实现常见的数据结构和算法

STL 的设计目标是高效、灵活和通用，使得开发者能够快速构建复杂的程序
